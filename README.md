# crossaut-switcher

1. download .exe
2. make a txt file in .exe folder with name "coordinates" 
3. write coordinates in the file, line by line
  - settings X
  - settings Y
  - movement settings X
  - movement settings Y
  - forward key X 
  - forward key Y
  - ok X 
  - ok Y 
4. save txt

usage: Shift + < , Shift + >
